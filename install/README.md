# Installation scripts

Some useful tool installation scripts

### Scripts:
- #### utils.sh:
    Contains useful utility functions
- #### nvim.sh:
    Installation script for nvim and the required packages

    ```bash
    sudo ./nvim.sh
    ```

    Execute the final command once everything is a success
